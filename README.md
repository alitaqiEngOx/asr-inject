# asr-inject
Compute aquifier-storage-and-recovery (ASR) efficiencies

Repository under active development.

## Installation
1- clone this repository;<br />
2- activate a clean virtual python environment with `python 3.12` (e.g., using `conda`);<br />
3- install `poetry`:
```
$ pip3 install poetry
```
4- navigate to the parent directory of your copy of this repository;<br />
5- install the package using `poetry`:
```
$ poetry install
```
